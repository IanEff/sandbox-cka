# Helm Install Local

## Instructions

There involves a local Helm chart located at **/opt/drill-chart**.

Install this chart into a new namespace called `jellyfish`.

The release name should be `my-jellyfish-release`.

## Verification

The release `my-jellyfish-release` must utilize the chart from `/opt/drill-chart`, be deployed in the `jellyfish` namespace, and its pods must be running.
